试一试 · 一键场景

状态机

  • Idle → 文件 + 语言齐 → Ready
  • Ready → Recognize → Recognizing
  • Recognizing → completion / cancel → Idle
  • Recognizing → onError → Failed

接口对应

  • audioFileURL ← 文件区选择
  • config.language ← Language 下拉，禁止 Auto/Unknown
  • config.audioFormat ← Audio Format 下拉，PCM 必须显式指定
  • onPartialResult → 流式文本区
  • completion → 日志区 + 状态切换
  • cancelRecognitionWithTaskId: ← Cancel 按钮
