三种内容流（按 contentType）

OriginalText · 累积值（覆盖式）
TranslatedText · 累积值（覆盖式）
AudioChunk · 增量字节（追加式）

设计要点

  • 语言条 Auto ⇄ 中文，会话中**禁用**互换 / 切换
  • 三种 content 字段语义不同，UI 必须按 contentType 分发
  • utteranceIndex 把同一句话的原文 / 译文 / 音频归一气泡
  • 同传只有 stop（含冲刷），刻意没有 cancel
  • 会话结束 completion(report, error)，report.utterances 直接持久化
  • endReason：UserStop / Interrupted / Error

事件 vs. 内容

  • onContent：高频流式（OriginalText / TranslatedText / AudioChunk）
  • onEvent：低频状态变更（SessionStarted / Utterance Started/Ended / LanguageDetected / TTS Playback Started/Ended / NetworkError / BleDisconnected）
  • 右下 ⚙️ 抽屉里的 Logs 区可看完整事件流
