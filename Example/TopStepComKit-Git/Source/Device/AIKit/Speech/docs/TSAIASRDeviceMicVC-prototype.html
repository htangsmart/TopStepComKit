TSAIASRDeviceMicVC · 交互原型

模拟「对设备麦克风音频做流式语音识别」的全流程。可在右侧主动注入「正常 / Cancel / 中断 / 网络错误 / 无设备」等场景。

当前状态

状态含义
Idle页面空闲，可启动
Starting已发请求，等首个 partial
Recognizing持续接收 partial 结果
Stopping已请求 stop，等 completion
Cancelling已请求 cancel，等清理
FinishedStop 成功，最终结果就绪
CancelledCancel 完成，结果作废
Failed启动失败 / 识别中报错 / Stop 报错
UnsupportedSDK 或设备不具备 AI 能力

SDK 接口映射

Start → recognizeSpeechWithDeviceMicConfig:onPartialResult:completion:
Stop  → stopDeviceMicRecognitionWithTaskId:（冲刷 → 最终结果）
Cancel → cancelRecognitionWithTaskId:（丢弃）
partial → TSAIASRPartialResult.text（累积，不是新增）
result  → TSAIASRDeviceMicResult（含录音文件）
Tips：原型默认走「场景①」。先选语言，点 Start 即可观察流式累积。Stop / Cancel 区别：Stop 拿最终结果与录音；Cancel 直接作废。
